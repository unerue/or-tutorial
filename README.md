`or-tutorial` is an operations research tutorial. 

#### Dependencies

`or-tutorial` requires:

* Python (>= 3.7.4)
* NumPy (>= 1.16.5)
* pandas (>= 0.25.1)
* PuLP (>= 1.6.9)
* GUROBI (>= 8.1.1)
* DEAP (>= 1.3.1)
